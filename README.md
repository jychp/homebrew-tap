# Jychp Tap

## How do I install these formulae?

`brew install jychp/tap/<formula>`

Or `brew tap jychp/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "jychp/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
